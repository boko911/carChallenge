# JULES CESAR CALCULATOR


Jules Cesar Calculator is a small react app that can add substract and multiply two Roman numerals.


# New Features!

  - You can add, substract and multiply two Roman numerals
  

# Upcoming Features!

  - JEST unit Test





### Installation

Install the dependencies and devDependencies and start the servers.

```sh
$ git clone https://github.com/boko911/carlChallenge.git
$ cd carlChallenge
$ npm install
$ npm start
```

Frontend url : http://localhost:3000
